# memory
Required for a University project, this game was coded in JavaFX using the NetBeans 8.2 IDE.
It follows the same rules as a classic memory game (flip a card, check the symbol, try to match the same symbols), while offering a local multiplayer experience, a UI, and some additionnal features such as a "comeback mechanic".
# Unused features
Some optional features (such as the ability to save and load a game state) could not be developped due to a lack of time. The code for these features are still visible and commented so that they can be observed and, possibly, completed and reused in the future.
# Comments
Due to the project being developped for a French university, all comments are in French. Despite this, all other elements use an English syntax, so the code is still comprehensible to English-speakers.
